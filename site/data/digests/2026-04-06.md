**Andrej Karpathy proposes "idea files" instead of sharing code**

Former Tesla AI director Andrej Karpathy shared a viral concept about how software development might change with AI agents. Instead of sharing actual code or apps, developers would share "idea files" — descriptions that AI agents could then customize and build for each person's specific needs.

Why it matters: This could fundamentally change open source development, shifting from sharing finished code to sharing concepts that AI can adapt and implement differently for each user.
